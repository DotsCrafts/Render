2Liquid Glass 设计语言 DESIGN

向 Apple Liquid Glass 靠拢的三层做法，为未来 AR / 穿戴显示预留「内容居中、玻璃悬浮」的空间语义：

  • 真实材质（窗口层）：macOS 窗口改为 titleBarStyle: hiddenInset + vibrancy: under-window + 透明背景 —— 去掉了截图里那条多余的 "Render" 标题栏（这就是标签「被盖住」的观感来源之一），红绿灯直接嵌进标签行，桌面透过半透明 chrome 形成真实玻璃。非 mac 平台自动回退实心窗口。
  • 材质令牌（tokens 层）：packages/ux-render/tokens.css 新增 --glass-bg / --glass-bg-strong / --glass-bg-soft / --glass-border / --glass-edge / --glass-blur / --glass-saturate / --shadow-glass(-pill)，暗色 + 亮色全套。chrome、agent 面板、模态、生成式卡片共享同一套 :root，一处定义全局生效。
  • 表面应用（chrome 层）：顶栏玻璃板（oklch(...) / 0.58 半透明 + 顶缘高光）、活动标签玻璃芯片、omnibox 胶囊、输入胶囊 blur(26px) saturate(1.4)、面板/模态玻璃化、Home 提示卡玻璃 chip、fallback 门户（portal.html）整页玻璃重绘。全部带 prefers-reduced-motion 守卫。
