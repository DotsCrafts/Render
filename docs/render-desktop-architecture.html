Render 本地桌面端架构

现状快照 · 基于 apps/desktop/src/main/index.ts 实际接线 · 2026-06-24

一句话心智模型：Render 自己就是浏览器。Electron 主进程是"可信的手的总线"， 托管一个窗口 —— 底层是 chrome 渲染器（标签栏 / 地址栏 / 右侧 agent 面板 / 底部浮动输入）， 上面叠真实的网页 WebContentsView。 大脑是跑在独立沙箱里的 codex；手是 CDP human-hand 驱动真实标签页； opencli 的 cookie/浏览器适配器经 bridge 在 Render 自己的 Chromium 里跑 —— 登录、抓取、生成页面全程不碰系统 Chrome。
大脑 Brain（推理 / codex） 手 Hands（CDP 驱动） 浏览器 Browser（Render 自身 + 会话） 隔离面 Isolated（Tier-2 阅后即焚）

2三条关键数据流

① 一次普通提问（文字 → UI 回答）
  1. 用户在浮动输入里发 prompt → 渲染器经 preload contextBridge → IPC submitPrompt。
  2. Agent Runtime 投给沙箱里的 codex app-server，开 turn_started。
  3. codex 推理，按需调工具（opencli 搜索 / render-open / render-artifact）。
  4. 结果以 AgentEvent 流回：item / delta / reasoning / ux(render·form·confirm·login) / artifact / turn_completed。
  5. 渲染器把 ux:render 的 json-render spec 画成真实 UI（不是纯聊天文本）。
② 登录某站并抓取（cookie/浏览器适配器）
  1. agent 跑 opencli <site> login --site-session persistent。
  2. opencli daemon 的 /ext 后端由 bridge 服务 → 在 Render 自身 Chromium 里铸造一个真实可见标签（落入当前会话组）。
  3. 用户在该标签里扫码 / 输密码 —— 大脑全程看不到凭据；cookie 落 persist:render。
  4. 用 whoami --site-session persistent 验证（不信 login 退出码；校验漂移按已登录处理）。
  5. 后续 search/read（同 persistent 会话）即已认证，数据流回成卡片。从不碰系统 Chrome。
③ Tier-2 生成式页面调后端（artifact → opencli）
  1. agent 写自包含 HTML，跑 render-artifact app.html --opencli "<site> <cmd>,…"。
  2. Runtime 解析 sentinel → tabs.createArtifact() 在隔离 artifact:<id> 分区起标签（无网 CSP + artifact-preload）。
  3. 页面调 window.renderArtifact.opencli(site, cmd, positional, args) → IPC artifactOpencli。
  4. 能力闸校验 allowlist + consent（原型期放行）→ runArtifactOpencli 用 buildArgv（位置参数在前）跑到 bridge profile。
  5. 读到用户登录态站点的真实数据，-f json 解析后回到页面渲染。页面阅后即焚。

3信任边界（为什么这样切）

渲染器：不可信展示面

sandbox:true + contextIsolation，只能经 preload 桥说话；任何顶层导航离开 chrome origin 都被拦成新标签 —— 面板里的链接永远替换不了应用 UI。

大脑：沙箱里的 codex

跑在独立沙箱；经 egress 代理上网，看不到 API key；看不到用户 cookie（cookie 在主进程 Chromium 的 persist:render，opencli 经 CDP 只取数据不取凭据）。

浏览器：Render 自身 + 会话

所有登录/浏览/生成都在 Render 自己的 Chromium；persist:render 持用户登录态（设备本地）。绝不跳系统 Chrome。

隔离面：Tier-2 最严

临时 artifact:<id> 分区（绝不用 persist:render）+ 无网 CSP，唯一出口是窄能力 renderArtifact（allowlist + consent）。关标签即销毁。
